# Claude Code 단축키

## 자주 쓰는 단축키

### 채팅 / 전반

| 단축키         | 설명                        |
| ----------- | ------------------------- |
| `Ctrl+R`    | 이전 대화 기록 검색               |
| `Ctrl+O`    | 대화 내역 보기/숨기기              |
| `Ctrl+T`    | Todo 패널 토글                |
| `Ctrl+S`    | 입력 중인 메시지 임시 저장           |
| `Ctrl+G`    | 외부 에디터로 메시지 작성            |
| `Shift+Tab` | 모드 전환 (auto / plan / ...) |

### 에이전트 실행 중

| 단축키 | 설명 |
|--------|------|
| `Ctrl+B` | 현재 작업을 백그라운드로 전환 |
| `Ctrl+X` → `Ctrl+K` | 모든 에이전트 강제 종료 |

### 권한 확인 화면에서

| 단축키 | 설명 |
|--------|------|
| `Ctrl+D` | 권한 디버그 정보 전환 |
| `Ctrl+E` | 권한 설명 펼치기/접기 |

→ `~/.claude/keybindings.json` 에서 원하는 키에 바인딩할 수 있다.
