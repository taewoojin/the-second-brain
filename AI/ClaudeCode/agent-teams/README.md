# Agent Teams 가이드

**한눈에 보기**

Agent Teams는 여러 Claude Code 세션을 팀으로 구성해서 병렬로 일하는 기능입니다. 팀 리더가 일을 분담하면, 팀원들이 각자 독립적으로 진행하면서 필요할 때 소통합니다.
